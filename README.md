# Gestión Operativa ADO

Dashboard web conectado a Google Sheets mediante Google Apps Script.

## Uso
1. Sube todo el contenido a GitHub.
2. Activa GitHub Pages apuntando a la rama principal.
3. Si cambias tu Apps Script, actualiza la URL en `js/api.js`.

## Estructura
- `index.html`
- `css/style.css`
- `css/dashboard.css`
- `css/responsive.css`
- `js/api.js`
- `js/dashboard.js`
- `js/charts.js`
- `js/login.js`
- `js/utils.js`
